NotedBot Commands

Normale Commands

ping
Zeigt dir den Ping und die Latenz des Bots an.
test
Der Bot schickt eine Nachricht zum Überprüfen, ob er im Channel ist.
serverinfo
Zeigt die Allgemeine Infos zum Server wo der Bot läuft an.

Info Commands

user
Gibt dir Infos über User an.
id
Gibt dir die IDs von Usern an.
uid
Sagt dir Welchen User die id Gehört.
chatters
Zeigt dir an, wie viele Chatter in deinem Channel sind.
isinchat
Sagt, ob der Chatter im Channel ist. (!isinchat "username"/!isinchat "username" "channel name").
roles
Zeigt dir wie viele Rollen du hast auf Twitch.
subage
Zeigt dir wie lange du schon Bei Jemanden Sub bist an.
mods
Zeigt dir die Mods des jeweiligen Kanals an. (!mods "channel name").
vips
Zeigt dir die VIPs des jeweiligen Kanals an. (!vips "channel name").
follow
Zeigt dir an, welcher Kanal wem folgt. (!follow "channel name").
founders
Zeigt dir die Founder des jeweiligen Kanals an. (!founder "channel name").

Vip Commands

fill
Schreib ein Langen Text mit (!fill "emotename").
